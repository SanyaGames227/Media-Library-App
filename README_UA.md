# 🎬 Media Library App

Потужний десктопний менеджер медіатеки на Python та PyQt5. Організовуйте фільми, серіали, ігри, книги — або будь-що інше — у чистому, швидкому інтерфейсі без інтернету.

> Ідея та дизайн: **sanyagames227** · Розробка: **Claude AI (Anthropic)**

---

## ✨ Можливості

### 📁 Гнучка організація
- **Папки та категорії** — групуйте медіа як завгодно, з іконкою та фоновим зображенням для кожної категорії
- **Необмежена кількість елементів** з обкладинкою, назвою, роком, оцінкою, нотатками, датою перегляду та іншим
- **Альтернативні назви** з підтримкою ключа сортування — шукайте за будь-яким псевдонімом, відображається основна назва
- **Вибране** — позначайте та фільтруйте миттєво

### 🔍 Потужний пошук і фільтрація
- **Пошук у реальному часі** за назвами та всіма користувацькими полями (Ctrl+F)
- **Користувацькі префікси пошуку** — задайте власні ключові слова фільтрації (наприклад `tag:horror`, `yw:2024`)
- **Фільтр за датою** — готові діапазони або довільний вибір дат
- **Перемикач «Лише вибране»**
- **Сортування** за назвою, роком, оцінкою, датою перегляду або вручну перетягуванням
- Пам'ять сортування по категоріях — кожна категорія запам'ятовує власне сортування

### 🏷️ Користувацькі поля
Створюйте власні поля даних для кожної категорії — будь-яка комбінація:
- Текст, Число, Дата, Прапорець, Відмітки (мульти-прапорець), Список вибору, Повзунок
- Показуйте поля у списку елементів, компактному режимі (емодзі/іконка) або в контекстному меню
- Використовуйте змінні полів у текстовому експорті як `{змінна}`

### 🎖️ Тір-листи
- Вбудована система тір-листів із повністю налаштовуваними наборами тірів

### 👥 Мультикористувацький режим
- Відстежуйте історію переглядів по користувачах — з ким дивився і скільки разів

### 📊 Статистика
- Графіки активності по місяцях та роках
- Середні оцінки, загальний час, перегляди наодинці та в компанії
- Розподіл за роками випуску
- Облік повторних переглядів

### 🖼️ Обкладинки
- Завантаження з файлу або вставка зображення/URL напряму
- Вбудований онлайн-пошук обкладинок (Bing)
- Налаштовувана якість JPEG для економії місця
- Інструмент очищення невикористаних зображень

### 📤 Експорт та імпорт
- **HTML-експорт** — з можливістю вбудувати зображення як base64 (один файл) або експортувати в папку
- **CSV/ZIP-експорт** — повний експорт даних з обкладинками, іконками та користувацькими полями
- **ZIP-імпорт** — відновлення з архіву з переглядом змін (бачите саме те, що буде додано)
- Текстовий експорт із повністю налаштовуваними шаблонами через змінні полів
- Пакетний експорт кількох категорій

### 🗒️ Нотатки
- Редактор нотаток для кожного елемента

### ⚙️ Налаштування та кастомізація
- **Підтримка мов** — English, Русский, Українська (та кастомні `.json`-файли мов)
- **HiDPI-масштабування** для QHD та 4K моніторів
- **Автоматичний щотижневий бекап** при запуску
- **Бекап при видаленні** — налаштовуване автозбереження при видаленні елементів, категорій, полів, тірів або користувачів
- **Обслуговування бази даних** — VACUUM-оптимізація з автоматичним бекапом
- Налаштовуваний формат дати
- Створення ярлика на робочому столі
